# Business Analyst Skill

## Description
This agent specializes in analyzing corporate operating models, competitive landscapes, and industry trends based on 10-K filings.

## Capabilities
- **Business Model Analysis**: Identifies revenue streams and cost structures.
- **Competitive Advantage**: Extracts information about 'moats' and market position.
- **Risk Assessment**: Summarizes competition-related risks.

## Usage
Provide the agent with a query related to the qualitative aspects of the company. The agent uses RAG to retrieve relevant sections from the ingested 10-K documents.
